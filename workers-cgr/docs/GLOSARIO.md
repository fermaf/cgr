# Glosario

- **RAW**: JSON original recibido desde CGR o dump Mongo.
- **Enrichment**: salida del LLM.
- **Vectorize**: upsert en Pinecone.
- **Cola**: lista de tareas asincronas.
- **Backfill**: proceso de recalculo/barrido para completar campos faltantes.
- **Crawl**: captura de dictamenes desde CGR.
- **CGR**: Contraloria General de la Republica.
